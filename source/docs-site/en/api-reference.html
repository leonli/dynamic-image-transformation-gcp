API reference

Request types

Detection order (first match wins):

  1. Default — path is valid base64 and decodes to JSON: GET /<base64(JSON)>
  2. Custom — REWRITE_MATCH_PATTERN and REWRITE_SUBSTITUTION are both set; the rewritten path is parsed as Thumbor
  3. Thumbor — path has no extension or ends in .jpg .jpeg .png .webp .tiff .tif .svg .gif .avif
  4. otherwise → 400 RequestTypeError

Default request schema

FieldTypeRequiredDescription
bucketstringNoSource bucket. Must be on the SOURCE_BUCKETS allow list (else 403 ImageBucket::CannotAccessBucket). Defaults to the first allow-list entry.
keystringYesObject key of the source image.
editsobjectNoEdit pipeline (below). Without edits and outputFormat the original bytes are returned.
outputFormatstringNoOne of jpg jpeg png webp tiff heif heic raw gif avif.
effortnumberNoWebP only. Truncated to an integer; valid 0–6, anything else falls back to 4.
headersobjectNoExtra response headers, filtered by the header deny list.

Pipeline order: resize is applied first (default fit=inside is injected when absent), then the remaining edits in object key order. Pass-through keys map directly onto the corresponding sharp calls.

Query parameters

ParameterEffect
signatureHMAC-SHA256 (hex) of path + sorted query (see signing).
expiresUTC YYYYMMDDTHHmmssZ; expired links are rejected.
formatOverrides toFormat.
fitOverrides resize.fit.
width / heightOverrides resize dimensions; 0 or empty clears the dimension.
rotateOverrides rotation; empty value means auto-orient.
flip / flopBooleans; "0", "false", "" are false.
grayscale / greyscaleBoolean, same falsy rules.

All other query parameters are stripped at the entry middleware; duplicates keep the last value; the surviving set is sorted by key (stable CDN cache keys).

Error codes

Errors are returned as JSON: {"status": n, "code": "…", "message": "…"} with Content-Type: application/json, CORS headers, and Cache-Control: max-age=10,public (4xx) or max-age=600,public (5xx).

StatusCodeWhen
400RequestTypeErrorPath matches no request type (extension not supported).
400DecodeRequest::CannotDecodeRequestBase64 path does not decode to valid JSON.
400DecodeRequest::CannotReadPathEmpty or unreadable path.
400QueryParameterParsingErrorQuery edit parameter could not be parsed.
400InvalidResizeExceptionResize width/height ≤ 0 after rounding.
400Crop::AreaOutOfBoundsManual crop rectangle exceeds the image.
400OverlayImageExceptionOverlay/watermark could not be applied.
400SmartCrop::FaceIndexOutOfRangefaceIndex ≥ number of detected faces.
400SmartCrop::PaddingOutOfBoundsPadding pushes the crop outside the image.
400AuthorizationQueryParametersErrorSignature required but missing (“Query-string requires the signature parameter.”).
400ImageRequestExpiryFormatexpires is not YYYYMMDDTHHmmssZ.
400ImageRequestExpiredexpires lies in the past.
403ImageBucket::CannotAccessBucketRequested bucket is not on the allow list.
403SignatureDoesNotMatchSignature verification failed.
404NoSuchKey“The image <key> does not exist or the request may not be base64 encoded properly.” Any GCS read failure maps here.
413TooLargeImageExceptionOnly with COMPAT_AWS_LIMITS=Yes: processed image exceeds 6 MB base64-encoded.
500SignatureValidationFailureSecret could not be read / unexpected signature error.
500SmartCrop::ErrorVision face detection failed unexpectedly.
500Rekognition::DetectModerationLabelsErrorVision SafeSearch call failed (AWS error name kept for compatibility).
500InternalError“Internal error. Please contact the system administrator.” — any unclassified failure.

With the fallback image enabled, error responses carry the fallback image body while keeping the original status code.

Header deny list

Custom headers entries matching the following (case-insensitive) are dropped:

  • Exact: authorization, connection, server, transfer-encoding, referrer-policy, permissions-policy, www-authenticate, proxy-authenticate, x-api-key, set-cookie
  • Prefixes: x-frame-*, x-content-*, x-xss-*, strict-transport-*, permissions-*, x-amz-*, x-amzn-*, access-control-*, cross-origin-*, content-*
